# smallfoot
A simpler version of [bigfoot.js](https://github.com/lemonmade/bigfoot) in ES6

## Usage

```html
<!-- Add JQuery -->
<script src="https://code.jquery.com/jquery-3.1.0.min.js"></script>
<!-- Add smallfoot -->
<script src="{path to smallfoot}">
  // Initialize
  $.smallfoot();
</script>
```
